-----------------------

I'm deeply interested in frontend and backend development, constantly building and optimizing web applications that deliver seamless user experiences. My journey revolves around React, Node.js, Express, and MongoDB, where I bring ideas to life through structured and efficient code. Some of my key projects: 🤔 Doubtroom-A doubt solving platform with focus on user metrics like strengths and weakness with faculty seperate profile 📝 Blogify – A dynamic blogging platform with a focus on intuitive UI and content management. 🤖 AI Chatbot – An interactive chatbot leveraging NLP for smart user interactions. 🌍 AQI Sensing Platform – A real-time air quality monitoring system integrating IoT and geospatial data. Excited to learn, collaborate, and build innovative tech solutions! Let’s connect and grow together! 🚀
